## 📦 Instalación y uso

### Requisitos

- Node.js >= 18.x
- Angular CLI >= 19.x

### Pasos

```bash
# 1. Clonar el repositorio
git clone https://github.com/claudiamoyano/Claudia-Portfolio.git

# 2. Instalar dependencias
npm install

# 3. Ejecutar en desarrollo
npm start

# 4. Construir para producción
npm run build
